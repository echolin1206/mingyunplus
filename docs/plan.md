# 命运+ (MingYunPlus) - 项目规划文档

## 项目概述

**域名**: mingyunplus.com  
**定位**: 有趣好玩的命运娱乐测试网站，让人放松解压  
**Slogan**: 探索命运的趣味，发现生活的惊喜

---

## 功能模块

### 核心功能
1. **命运测试** - 趣味性格/命运测试，多维度分析
2. **塔罗占卜** - 经典78张塔罗牌，支持多种牌阵
3. **紫微星盘** - 根据生辰生成紫微斗数命盘
4. **星座运势** - 12星座日/周/月/年运势
5. **流年运势** - 根据生肖/八字看年度运势
6. **每日宜忌** - 今日宜忌明细 + 幸运色 + 祈运语

### 商业功能
- **单次测算**: ¥9.9/次
- **终身会员**: ¥39/永久
- **支付方式**: 企业支付宝
- **用户系统**: 微信授权登录 / 手机号登录

## 技术架构

```
mingyunplus/
├── public/          # 静态资源
│   ├── css/         # 样式文件
│   ├── js/          # 前端脚本
│   └── images/      # 图片资源
├── server/          # 后端服务
│   ├── routes/      # API路由
│   ├── models/      # 数据模型
│   ├── middleware/  # 中间件
│   └── utils/       # 工具函数
├── config/          # 配置文件
├── docs/            # 文档
└── package.json     # 依赖管理
```

### 技术栈
- **后端**: Node.js + Express
- **数据库**: SQLite (轻量，无需配置)
- **前端**: 原生 HTML5 + CSS3 + JavaScript
- **支付**: 支付宝企业版 SDK
- **部署**: PM2 + Nginx (阿里云)

## 支付流程

1. 用户选择服务 → 点击"立即测算"
2. 检查用户VIP状态 → 终身会员直接展示结果
3. 非VIP → 展示支付弹窗（9.9元/次 或 39元终身）
4. 选择支付方式 → 调用支付宝统一下单API
5. 生成支付二维码 → 用户扫码支付
6. 支付宝异步通知 → 更新订单状态
7. 前端轮询/websocket通知 → 展示结果

## 部署方案

### 阿里云配置
- **服务器**: ECS 2核4G (CentOS/Ubuntu)
- **域名**: mingyunplus.com (已备案)
- **SSL**: 阿里云免费SSL证书
- **Nginx**: 反向代理 + 静态文件服务
- **PM2**: Node.js进程管理

### 部署步骤
1. 服务器安装 Node.js + PM2 + Nginx + SQLite
2. 上传代码，安装依赖
3. 配置 Nginx 反向代理到 3000 端口
4. 配置 SSL 证书
5. PM2 启动应用
6. 配置支付宝回调域名

- 所有API接口验证签名
